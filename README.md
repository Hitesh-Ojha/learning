# Learning Journey — Data Science, ML Foundations & DSA

This repository documents my structured learning path in data analytics/ML foundations and core problem-solving through DSA.

The goal is to build a solid base in Python, NumPy, Pandas, and data visualization — then move into EDA projects, BI tools (Excel, Power BI), and SQL — while independently strengthening algorithmic thinking through consistent DSA practice.

---

## Data Analytics Track

### NumPy
Fundamentals of numerical computing with NumPy:
- Array creation — `array`, `zeros`, `ones`, `identity`, `arange`, `linspace`, `copy`
- Array attributes — `shape`, `ndim`, `size`, `itemsize`, `dtype`
- Type conversion with `astype`
- Memory and performance comparisons vs native Python lists
- Indexing, slicing, and boolean indexing
- Element-wise and scalar operations, dot product
- Aggregate functions — `min`, `max`, `mean`, `std`, `median`, `sum`
- Trigonometric and exponential functions
- Reshaping — `reshape`, `ravel`, `transpose`
- Splitting and stacking — `hstack`, `vstack`, `hsplit`, `vsplit`
- Random array generation with `np.random`
- Broadcasting for arrays of different shapes

### Pandas
Data manipulation and analysis with Pandas:
- Reading CSV files with `read_csv`
- Exploring data — `head`, `tail`, `shape`, `info`, `describe`
- Column selection and Series creation
- Boolean masking for filtering rows
- Element-wise access with `.loc` / `.iloc`
- Grouping with `groupby`, `value_counts`
- Sorting with `sort_values`
- Index manipulation with `set_index`
- Plotting integration with Matplotlib

### Matplotlib
Data visualization fundamentals with Matplotlib:
- Line plots — markers, colors, linestyles, linewidth, labels, legend
- Bar plots — grouped bars with width offsets, `xticks` rotation
- Horizontal bar plots with `barh`
- Pie charts — `explode`, `autopct`, `shadow`, `wedgeprops`
- Stack plots with `stackplot`
- Filled area plots with `fill_between`
- Subplots for multi-panel figures
- Scatter plots from CSV data
- Date formatting for time-series plots
- Real-time animated plots with `FuncAnimation`
- Reading data from CSV with `csv.DictReader` and pandas

### Seaborn
Statistical data visualization built on Matplotlib:
- Relational plots — `relplot` with `scatter` and `line` kinds
- Hue, style, and size parameters for multi-dimensional encoding
- `scatterplot` for detailed scatter visualization
- Categorical plots — `catplot` with `strip`, `swarm`, `box`, `violin`, `bar`, `count` kinds
- `swarmplot` and `boxplot` directly
- Distribution plots — `distplot`, `kdeplot`
- `heatmap` for correlation matrices
- `pairplot` and `jointplot` for multivariate analysis
- `FacetGrid` for faceted subplots

Linear algebra, probability, and statistics fundamentals for ML.
